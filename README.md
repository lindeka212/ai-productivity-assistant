# AI Workplace Productivity Assistant

## Project Overview

The **AI Workplace Productivity Assistant** is a modern web application that helps professionals automate everyday workplace tasks using AI. It provides a unified dashboard where users can draft emails, summarize meetings, plan tasks, conduct research, and chat with an AI assistant — all from a single, responsive interface.

The app is designed around a clean SaaS-style dashboard with sidebar navigation, structured prompts behind the scenes, and editable AI outputs so users always stay in control of what the AI produces. A responsible-AI disclaimer is shown across tools to remind users to review generated content before use.

## Features

- **Smart Email Generator** — Draft professional emails from a short brief (recipient, tone, key points).
- **Meeting Notes Summarizer** — Turn raw meeting transcripts or notes into structured summaries with decisions and action items.
- **AI Task Planner** — Break down goals into prioritized, actionable task lists with suggested timelines.
- **AI Research Assistant** — Generate structured research briefs on any topic with key findings and sources to verify.
- **AI Chatbot Interface** — Multi-turn conversational assistant for open-ended workplace questions.
- **Modern Dashboard UI** with collapsible sidebar navigation.
- **Fully Responsive Design** — works on mobile, tablet, and desktop.
- **Editable AI Outputs** — every generated response can be edited inline before exporting or copying.
- **Streaming Responses** — token-by-token AI output via Server-Sent Events.
- **Responsible AI Disclaimer** displayed on every tool.

## Tools Used

### Frontend
- **React 19** + **TypeScript**
- **TanStack Start** (full-stack React framework) with **TanStack Router**
- **Vite 7** as build tool
- **Tailwind CSS v4** for styling
- **shadcn/ui** components
- **Space Grotesk** typography with a custom `oklch`-based design system

### Backend
- **Lovable Cloud** (managed backend) for database, auth, and edge functions
- **TanStack Start Server Functions** for app-internal server logic
- **Edge Function** (`ai-assistant`) routing AI requests and streaming SSE responses

### AI
- **Lovable AI Gateway** using `google/gemini-3-flash-preview`
- Structured system prompts per tool (email, meeting, planner, research, chat)

## Setup Instructions

### Prerequisites
- [Bun](https://bun.sh) (recommended) or Node.js 20+
- A Lovable Cloud–enabled project (backend is auto-provisioned)

### Local Development

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd ai-workplace-productivity-assistant
   ```

2. **Install dependencies**
   ```bash
   bun install
   ```

3. **Environment variables**

   The `.env` file is auto-generated by Lovable Cloud and contains:
   ```
   VITE_SUPABASE_URL=...
   VITE_SUPABASE_PUBLISHABLE_KEY=...
   VITE_SUPABASE_PROJECT_ID=...
   ```
   No manual API keys are required — the Lovable AI Gateway handles model access on the server.

4. **Start the dev server**
   ```bash
   bun run dev
   ```
   The app will be available at `http://localhost:8080`.

5. **Build for production**
   ```bash
   bun run build
   ```

## Responsible AI

This application uses generative AI models that can occasionally produce inaccurate or biased content. Always review and edit AI-generated output before sharing it externally. Do not submit confidential or personally identifiable information into prompts.
